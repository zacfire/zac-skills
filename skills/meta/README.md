# Meta 板块

跨领域的工具 skill——不属于任何具体领域，但各领域都要用。

## 当前 Skills

| Skill | 用途 | 出处 |
|---|---|---|
| read | 抓取 URL 或 PDF 为 Markdown | Waza（原版） |
| learn | 深度研究到成稿（六阶段） | Waza（原版） |

## 设计理念

meta 工具是"原子能力"——不做判断，只做数据转换或信息采集。

其他 skill 经常会调用 meta：
- research → read（抓链接）→ 分析
- official-blog → read（抓参考文章）→ learn（深入理解）→ 写稿

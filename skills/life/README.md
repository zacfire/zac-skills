# Life 板块

个人体验/消费沉淀的方法论。**不是娱乐本身，是娱乐之后的思考。**

## 待建 Skills

| Skill | 用途 |
|---|---|
| reading | 读完一本书后的批判性记录（不是摘抄，是对话） |
| watching | 看完电影/剧/播客后的反思 |
| reflection | 周/月/季度复盘 |

## 设计理念

life 板块的 skill 不应该是"百科式笔记模板"，应该是 **"Zac 式的思考结构"**：
- 不记"书讲了什么"，记"我不同意什么"
- 不记"电影剧情"，记"什么触动了我"
- 不记"这周做了什么"，记"哪些判断错了"
